Desarrollo Web Orientado a Servicios

Este portafolio presenta las evidencias del desarrollo de una aplicación web orientada a servicios, aplicando conceptos de arquitectura SOA y el uso estratégico de APIs modernas.

Objetivo

Dominar el desarrollo de aplicaciones web con arquitectura orientada a servicios

Aprendizaje

Implementación práctica de APIs y servicios web profesionales
